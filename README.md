# task-tracker-pro

Make me a simple task tracker with the ability to have tasks and then group them under projects. I should also be able to attach images to a task, set the due date, a status (backlog, in-progress, review, done). Add any other metrics you might suggest.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/task-tracker-pro.git
cd task-tracker-pro
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
